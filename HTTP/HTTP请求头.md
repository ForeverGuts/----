# 对于服务器
通过判定请求头来：
- 判定请求是否合法 爬虫需要添加多个请求头来模拟真实的网页请求
- 判断用户是否登录 一般通过==Cookie==这个特殊的请求头的内容来判定 只要我们找到合理的cookie，服务器就认为我们通过了
# 重要组成
1. User-Agent
![[Pasted image 20251226150844.png]]
代表发起访问的什么浏览器，如果不写，基本会被判定为爬虫，直接被拒绝
2. Cookie 

里面记录了登录信息，或者上次请求服务端设置的信息，也是常用的反爬判定点
3. Referer
![[Pasted image 20251226150917.png]]
表示这次请求是从哪里点过来的，有的网站不允许你直接访问某个网页，必须从它的指定网页进去，这时候设置Referer值，就是模拟从别的网页点过来的情况

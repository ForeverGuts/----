# GET请求参数
GET请求的参数都是放在URL中，网页可以==直观==看见
![[Pasted image 20251226144745.png]]
请求参数是以键值对形式出现，一般形式为:
![[Pasted image 20251226144554.png]]
- 网址后面紧跟者是一个问号
- 问号的后面是一对对参数，形式为: 参数1= Value
- 多个参数之间用&隔开

# Post请求参数
- POS请求的参数放在==请求体==中
- 网页上看不见
- 但可以通过Chrome开发者工具或抓包工具看见
